# Freelancer Portfolio Website

A simple, modern, and responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean and professional design
- **Smooth Scrolling** - Enhanced navigation experience
- **Interactive Elements** - Hover effects and animations
- **Contact Form** - Ready-to-use contact form
- **Project Showcase** - Display your projects with descriptions and tech stacks

## Sections

1. **Hero Section** - Introduction and call-to-action buttons
2. **About Section** - Personal information and statistics
3. **Skills Section** - Your technical skills and expertise
4. **Projects Section** - Showcase of your work (6 example projects included)
5. **Contact Section** - Contact information and form

## Getting Started

1. Open `index.html` in your web browser
2. Customize the content with your own information:
   - Replace "Your Name" with your actual name
   - Update email, phone, and location in the contact section
   - Add your social media links
   - Update project details with your actual projects
   - Modify skills and statistics

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #6366f1;
    --secondary-color: #8b5cf6;
    /* ... */
}
```

### Projects
Update the project cards in `index.html` with your own projects. Each project card includes:
- Project image/icon
- Project title
- Description
- Technology tags
- Links to live demo and GitHub

### Contact Form
The contact form is currently set up with basic validation. To make it functional, you'll need to:
- Connect it to a backend service (e.g., EmailJS, Formspree, or your own server)
- Or modify the JavaScript in `script.js` to handle form submission
